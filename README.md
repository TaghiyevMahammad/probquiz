# Prob Quiz - Numerical & Multiple Choice Questions

An interactive quiz game built with HTML, CSS, and JavaScript. Users can select a range of questions and the number of questions they want to attempt. Questions and answers are loaded from a `.txt` file.

## Live Demo

Play online: [Prob Quiz Demo](https://probquiz.netlify.app/)

## Features

- ✅ Load questions and answers from `prob.txt`.
- ✅ Supports numerical and multiple-choice questions.
- ✅ Randomized questions and answer options.
- ✅ Instant feedback on correct/incorrect answers.
- ✅ Score tracking at the end of the quiz.
- ✅ Responsive and clean user interface.

## How to Use

1. Open `index.html` in a browser.
2. Enter:
   - **Başlanğıc sual nömrəsi:** starting question number.
   - **Son sual nömrəsi:** ending question number.
   - **Sual sayı:** how many questions to attempt.
3. Click **Sınağa Başla** to start the quiz.
4. Click on an answer and then **Cavabla** to check your answer.
5. Click **Növbəti** to go to the next question.
6. At the end, your total score will be displayed.

## How It Works

- Questions are read from `prob.txt`.
- Correct answers are marked with `√` in the file.
- Other options are marked with `•`.
- The game shuffles questions and answer options for variety.
